# Project Features

Amr Samir Edris's portfolio site integrates professional event metrics-focused UI design with self-contained administrative and analytics dashboards.

---

## 1. Visual & Typography System (Apple Design Style)
* **Apple Design Aesthetic:** Clean margins, system typography, liquid glassmorphic overlays, and Space Gray layouts designed to resemble Apple product landing pages.
* **Apple Obsidian & Silver Themes:** Built-in Light (Apple Silver/Chalk) and Dark (Apple Obsidian/Space Gray) mode support, with local storage memory preservation.
* **Asymmetric Bento Grids:** Experiences and project credentials automatically align into a custom asymmetric bento grid.

---

## 2. Interactive Document Cabinet & Features
* **Document Drawers:** Minimalist drawers containing Curriculum Vitae and Event Portfolio cards. Toggling slides down an inline browser PDF viewer on desktop, with responsive link stream fallbacks on mobile.
* **Dynamic Contact Card (vCard):** Generates and downloads a `.vcf` contact file on clicking the Save Contact Card button.
* **Interactive Career Timeline Slider:** An Apple-style range slider widget allowing visitors to filter project cards dynamically by specific years (2021–2026) or select "All".

---

## 3. Availability Glowing Badge & CMS
* **Glowing Badge:** Displays a glowing availability dot (Green = Available, Amber = Open for Hiring, Red = Fully Booked) next to the hero title.
* **CMS status control:** Admin can set the status badge value directly from the general CMS settings panel.

---

## 4. Vercel Webhook Integration
* **Production Rebuild Trigger:** Admin can input a Vercel Deploy Hook and trigger production website rebuilds directly from the dashboard panel with a single click.

---

## 5. Analytics & Dashboard Console
* **Console UI:** Styled to resemble macOS System Settings.
* **KPI Metrics:** Track total unique visitors, page views, and download counts.
* **Visual Visibility Tracker:** Logs exact stay durations (in seconds) per major site section (Hero, Experience, CV Viewer, Portfolio Viewer) without using third-party trackers.
* **Visual Charts:** Line charts (daily visits) and bar charts (average stay times per section) rendered via Chart.js with theme-matching grid adjustments.
